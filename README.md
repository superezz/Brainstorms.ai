# 🚀 Exam Notes AI

AI-powered platform to generate **exam-oriented notes, project notes, charts, and downloadable PDFs instantly**.

---

## ✨ Features

* 🎁 50 Free Credits on signup
* ⚡ Generate notes instantly using AI
* 📊 Create charts & graphs
* 📄 Download clean PDF notes
* 🔐 Google Authentication
* 🎨 Modern animated UI (Framer Motion + Tailwind)

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🎬 Framer Motion
* 🔑 Google Auth (planned / in progress)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/exam-notes-ai.git
cd exam-notes-ai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```
